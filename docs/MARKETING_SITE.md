# Nyabag Marketing Site

This project is a focused public marketing repository.

Keep:

- public marketing pages
- blog and SEO content
- legal/contact pages
- brand assets
- early-access signup storage in `early_access_signups`
- deployment metadata such as sitemap, robots, and llms.txt

Do not add product-app implementation here. Bookmark search, canvas, extension capture, Telegram capture, user profiles, admin workflows, background processors, and app Supabase schema belong in the separate Nyabag app repository.
