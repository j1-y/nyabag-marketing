# Product

Nyabag is a desktop-first design memory system for collecting, organizing, and rediscovering visual inspiration.

## What the product is

- A bookmark workspace for designers and builders.
- A visual library for screenshots, palettes, fonts, summaries, and tags.
- An infinite canvas for notes, media, links, and social references.
- A structured workspace with folders, Design DNA, profile, onboarding, and admin support.

## Primary user loop

1. Save a bookmark.
2. Let the system enrich it with metadata, screenshots, and design data.
3. Organize it in folders or use it as source material for Design DNA.
4. Move inspiration onto the canvas for synthesis and planning.
5. Rediscover it through search, filters, and memory-oriented views.

## Audience

- Designers working on large screens.
- Founders and product builders collecting visual references.
- Future AI agents that need a stable, documented workspace model.

## Product boundaries

- Desktop is the full experience.
- Mobile is capture-first and intentionally limited.
- This repo includes developer infrastructure and internal tooling as well as user-facing surfaces.
- The AI memory layer is not a product feature.

## Current surface areas

- Bookmarks dashboard
- Folder hierarchy and inbox
- Bookmark detail pages
- Design DNA extraction and review
- Infinite canvas and media notes
- Mobile URL capture
- Profile and onboarding
- Public landing, blog, privacy, terms, and contact pages
- Admin views and extension endpoints
